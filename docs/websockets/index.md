# WebSocket

NestJS 为 WebSocket 提供了全面的支持，让您能够构建实时应用程序。

## 主要特性

- **Socket.IO** - 支持 Socket.IO 库
- **原生 WebSocket** - 原生 WebSocket 协议支持
- **命名空间** - 逻辑分组连接
- **房间** - 动态分组用户
- **适配器** - 多实例和集群支持
- **守卫和拦截器** - WebSocket 层面的安全和拦截

## 应用场景

- **实时聊天** - 即时消息应用
- **实时通知** - 推送通知系统
- **协作应用** - 多用户协作编辑
- **游戏** - 实时多人游戏
- **监控面板** - 实时数据展示
